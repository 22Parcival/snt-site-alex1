# 🌐 Guide d'installation - Site [Nom du Projet]

Ce guide vous explique comment récupérer et lancer le site internet sur votre ordinateur en quelques secondes.

## 🚀 Comment lancer le site ?

### 1. Téléchargement
* Cliquez sur le bouton vert **"Code"** en haut à droite de cette page.
* Choisissez **"Download ZIP"**.
* Une fois le téléchargement terminé, faites un clic droit sur le fichier `.zip` et choisissez **"Extraire tout"**.

### 2. Lancement
* Ouvrez le dossier décompressé.
* Cherchez le fichier nommé **`index.html`**.
* Faites un double-clic dessus : le site s'ouvrira automatiquement dans votre navigateur (Chrome, Firefox, Edge, etc.).
